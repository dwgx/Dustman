# 灰尘男人 (Dust Man)

一个基于PyQt5和Fluent Design的Windows优化工具，提供一键清理、优化、修复和文件管理功能。适用于Windows系统，支持网络修复、垃圾清理、性能优化等操作。

## 功能概述

- **网络修复**: 修复网络问题、PAC代理修复、协议修复、DNS管理（设置、刷新、速度测试）。
- **电脑清理**: 扫描和清理垃圾文件、清空回收站、清理浏览器缓存、Windows更新临时文件、DirectX着色器缓存。
- **电脑优化**: 优化RAM、清理后台进程、CPU/线程优化、设置卓越性能电源计划、显存优化、一键全面优化。实时监控CPU和内存使用率。
- **文件管理**: 浏览文件系统、删除/重命名文件、编辑文本文件、计算文件夹大小、在资源管理器中打开。
- **操作日志**: 记录所有操作细节。
- **设置**: 主题切换、通知设置、快捷键自定义。
- **程序信息**: 版本和描述。

**注意**: 
- 部分功能（如网络修复、深度清理）需要以管理员身份运行程序。
- 该工具仅适用于Windows系统（依赖特定命令如`netsh`、`powercfg`、`ipconfig`）。
- 清理和优化操作可能导致数据丢失，请备份重要文件。

## 安装指南

### 先决条件
- Python 3.8+（推荐3.12）。
- Windows操作系统（测试于Windows 10/11）。

### 步骤
1. 克隆或下载仓库：
2. https://github.com/dwgx/Dustman
3. 您需要安装的库
4. PyQt5==5.15.10 
5. PyQt-Fluent-Widgets==1.7.0 
6. psutil==6.0.0
7. 运行Main.py
